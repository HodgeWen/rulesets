# 机场推荐研究报告

当前日期：2026-04-03  
检索区间：2025-07-01 至 2026-04-03

## 先说结论

按 `ariport.md` 的标准，我**没有硬凑满 8+8**。

原因很直接：

- 2025 下半年到 2026 一季度，机场圈营销污染比以前更重，很多“评测”本质上是带 aff 的导购页。
- 真正有参考价值的高权重信息，主要还是 V2EX 长期用户反馈、Telegram 频道维护记录、GitHub 跑路/预警 issue。
- 满足“口碑没明显断崖、近阶段没大面积翻车、价格基本可交叉验证”的对象，实际没有多到可以放心写满 16 家。

这次筛选最看重 4 条：

1. 2025-07-01 之后是否还有持续维护、持续被真实用户使用的迹象。
2. 是否出现过大面积不可用、客服消失、官方群静默、接盘/维权信号。
3. 价格是否能从官网、用户讨论、第三方导航至少两路交叉验证。
4. 是否存在“AI/流媒体能用，但晚高峰、UDP、游戏、语音明显拉跨”的结构性短板。

## 一线高端机场

本轮只有 5 家达到我愿意放进“一线高端”的门槛。

### 1. Nexitally

- 名称：Nexitally
- 类别定位：一线高端，偏重流媒体解锁、日常稳定性、商务/重度用户
- 价格区间：高价位，公开可见样本里 `Smart Air 200G/年约 ¥624`，更高档位继续上探
- 核心优势：运营时间长；高端定位明确；V2EX 里有多位多年用户仍在续费；流媒体与 AI 可用性口碑仍处在头部
- 已知缺点：贵；2026-03 出现过一轮明显卡顿/被打；UDP 和游戏场景口碑一般；“万人骑 IP”对 AI 纯净度不是最佳
- 信息来源可信度评估：`S 为主，B 为辅`
  - S 级：V2EX 2025-06、2026-01、2026-03 的真实用户讨论里，既有多年续费正反馈，也有对 UDP/晚高峰卡顿的负反馈
  - S 级：官方 Telegram 镜像页面仍有持续公告与联络邮箱
  - B 级：官网用于确认成立时间、线路定位、节点覆盖
  - 结论：可信度高，但“高端不等于无波动”，近期稳定性不是无条件碾压

### 2. AmyTelecom

- 名称：AmyTelecom
- 类别定位：一线偏高性价比，常被视作奶昔系分站
- 价格区间：公开可见低门槛样本为 `2 元体验档起`，正式订阅整体低于 Nexitally
- 核心优势：长期被放在奶昔系替代/分站序列里讨论；价格明显低于奶昔主站；解锁与日常速度口碑普遍不差
- 已知缺点：UDP/游戏表现偏弱；V2EX 明确有人提到不提供 Trojan、Discord/游戏语音体验不稳；更适合网页、视频、AI 日常场景，不适合把它当游戏加速器
- 信息来源可信度评估：`S 为主，B 为辅`
  - S 级：V2EX 2025-06 对奶昔系/amy 的 UDP 短板有明确负反馈；2026-01 仍有人把 amy 用作前置机场
  - S 级：Telegram 群镜像在 2025-07 仍有活跃更新时间
  - B 级：官网可见“2 元套餐”信息，但详细价格透明度不如 TAG、Imm 这类更好交叉验证的对象
  - 结论：属于“主力可用，但场景边界清楚”的一线，不是全能型

### 3. TAG

- 名称：TAG
- 类别定位：一线高端，偏 AI、家宽出口、地区覆盖广
- 价格区间：公开样本可见 `¥99/月 500G`，`¥160/季（月 250G）`
- 核心优势：近阶段口碑重点集中在“家宽比较多”“AI 更稳”；地区覆盖和落地类型丰富；适合对 OpenAI/Gemini/区域解锁更敏感的用户
- 已知缺点：不算便宜；高流量下载党不一定觉得值；公开用户样本量少于奶昔系
- 信息来源可信度评估：`S 中高，B 中`
  - S 级：V2EX 2025-06、2026-03 都有直接用户提到 TAG 家宽和 AI 体验
  - S/B 级：最近一周 V2EX 新帖仍有人把 TAG 列为主力推荐
  - B 级：价格主要来自 Telegram 频道镜像和价格页镜像，存在渠道镜像偏差
  - 结论：适合把“AI 可用性”和“家宽出口”放在优先级前面的用户

### 4. ImmTelecom

- 名称：ImmTelecom
- 类别定位：一线高端，偏稳定专线、流媒体、轻团队共享
- 价格区间：公开可见样本为 `¥130/季 200G`、`¥72/月 500G`、`¥144/月 1000G`
- 核心优势：2025-2026 仍持续被讨论；价格通常低于 Nexitally 顶档；流媒体、AI、主流地区体验整体稳定；部分讨论提到有家宽落地
- 已知缺点：依然不便宜；有人反馈订阅更新偶尔失败；也有人买一周就弃用，说明体验存在人群差异
- 信息来源可信度评估：`S 中高，A/B 中`
  - S 级：V2EX 2025-03、2025-06、2026-01 均能看到用户实际使用反馈
  - A/B 级：第三方评测与机场导航对价格、节点结构给出的信息较一致，但这些来源存在推广成分，只能辅助
  - 结论：仍然在一线序列里，但没有强到可以无脑盲买

### 5. Kuromis

- 名称：Kuromis（库洛米）
- 类别定位：一线高端的次旗舰/分站型选择
- 价格区间：公开样本可见 `最低约 ¥408/年，50G/月`
- 核心优势：经常被放进奶昔系或“唯云四杰”语境里讨论；价格显著低于奶昔主站；Telegram 渠道在 2025-07 之后仍有维护记录
- 已知缺点：比多数性价比机场贵；公开真实用户样本明显少于 Nexitally/Amy/Imm；价格不算亲民
- 信息来源可信度评估：`S 中，B 中`
  - S 级：Telegram 频道镜像能看到官网、BOT、维护通知仍在持续更新
  - S 级：V2EX 用户页与相关讨论中仍把 Kuromis 放进头部高质量机场序列
  - B 级：最低价位主要来自导航/博客类页面，只能作为价格参考
  - 结论：更像“便宜一点的高端分站”，不是纯性价比机场

## 准一线高性价比机场

本轮只有 5 家达到我愿意放进“准一线高性价比”的门槛。

### 1. iKuuu

- 名称：iKuuu
- 类别定位：准一线高性价比，适合轻中度主用或重度备用
- 价格区间：`¥15/月 300G`；按量包 `¥25/300G`、`¥60/1000G`
- 核心优势：便宜；免费流量策略让很多人先试后买；V2EX 里 2025-07 仍有人直接评价“小众但是比较稳”；按量包对轻度用户友好
- 已知缺点：域名经常换；官方自己都承认 2025 有被打/部分节点失效情形；高峰期不是一线体验
- 信息来源可信度评估：`S 中，B 中`
  - S 级：V2EX 2025-07 有多条真实使用反馈
  - B 级：官网可确认价格、免费流量、按量包，但官方会夹带引流别家内容，需打折看
  - 结论：高性价比成立，但不建议把它神化成“平替一线高端”

### 2. V2NY

- 名称：V2NY
- 类别定位：准一线高性价比，适合低预算长期用
- 价格区间：公开样本集中在 `年付约 ¥100 左右的入门档`，也有按量方案
- 核心优势：V2EX 里出现了“用了三四年”“一年多可按月和按量”的持续样本；低价位下稳定性反馈不差
- 已知缺点：节点规模、售后、入口信息都比较低调；公开高质量外部测评很少；价格与官网地址容易被镜像站污染
- 信息来源可信度评估：`S 中，B 低`
  - S 级：V2EX 2025-07 和 2026-01 都能找到持续使用反馈
  - B 级：缺少靠谱官方价格页快照，价格区间主要靠用户口径交叉
  - 结论：更像“老用户口口相传的小场”，不是信息透明的一线品牌

### 3. XFLTD

- 名称：XFLTD（养鸡场）
- 类别定位：准一线高性价比，偏低价大流量和基础专线升级
- 价格区间：公开样本可见 `¥7/月 150G`，也有 `约 ¥20/100G` 的按量口径
- 核心优势：便宜；2025 年仍有人在 V2EX 直接说“一年多了很便宜也稳定”；Telegram 频道镜像能看到域名切换、维护公告，说明不是完全失联式运营
- 已知缺点：公开口碑两极分化；稳定性上限有限；更适合预算型用户，不适合把它当绝对主力一线
- 信息来源可信度评估：`S 中，B 中`
  - S 级：V2EX 2025-02 的直接用户反馈
  - S 级：Telegram 镜像里能看到 2025 年维护和域名切换公告
  - B 级：价格更多来自导航/博客类页面，营销污染较重
  - 结论：可以进准一线，但只能放在“便宜且能用”这一档，不是“高端体验低价平替”

### 4. 蜂鸟云

- 名称：蜂鸟云
- 类别定位：准一线高性价比，偏基础中转、低门槛月付
- 价格区间：公开样本较集中，为 `¥10/月 100G`
- 核心优势：便宜直观；V2EX 2025-03、2025-04、2025-06、2025-07 都能看到它被反复提到；有用户明确说“用了一年多”“特殊时期依然坚挺”
- 已知缺点：正面样本里同一批活跃推荐者重复出现，存在推广污染嫌疑；高质量独立讨论不足；不适合盲冲长付
- 信息来源可信度评估：`S 中低，B 中`
  - S 级：V2EX 至少有多次提及，且不止一个账号
  - B 级：主要信息仍以带测速图/带链接推荐为主，营销成分偏重
  - 结论：能进准一线，但置信度明显低于 iKuuu 和 V2NY

### 5. 魔戒

- 名称：魔戒（Mojie）
- 类别定位：准一线高性价比，偏按量付费、轻中度长期备用
- 价格区间：公开样本可见 `¥14.9/130G`，主打不限时按量
- 核心优势：按量不限时是它最大的卖点；V2EX 有“按量付费 2 年了挺稳”“十几块钱用了两年”的直接反馈；很适合不想被月度流量绑死的人
- 已知缺点：更适合轻中度；高峰期与顶级专线不是同一档；公开独立高质量测评少
- 信息来源可信度评估：`S 中，B 中`
  - S 级：V2EX 2025-03、2025-07 有实际用户反馈
  - B 级：价格主要来自导航与镜像信息页
  - 结论：如果你是轻中度、以“低总成本+按量保留”为主，它比很多月付场更合理

## 未入选但值得警惕的机场

### 1. Dler

- 未入选原因：2026-02 V2EX 直接出现“大面积节点不可用、客服消失、传言被抓”的强烈预警，不符合当前稳健筛选标准

### 2. SSRDOG

- 未入选原因：2025-07 GitHub 有连续一周节点红色失联预警；后续官方页面自己也提到 2025-11 曾发生 DNS 污染事件
- 说明：不是铁证“已跑路”，但至少已经不适合放进“靠谱推荐名单”

### 3. 一元机场

- 未入选原因：2025-11 V2EX 已出现维权帖，讨论里有“备用都没法用”“测速低延迟但实际不通”的集中负反馈
- 说明：典型的极低价备用场，不适合再被当作“靠谱主力”

### 4. 一云梯

- 未入选原因：2026 年的导航/汇总类页面已出现“软跑路”描述
- 说明：这条证据权重不如 V2EX/GitHub，但在缺少反证的情况下，我不会把它放进推荐

### 5. FlowerCloud（花云）

- 未入选原因：2025-08 有 Reddit 反馈“无法稳定连接 Google”；2026-03 V2EX 又出现“速度一上去就被摁下来”的近期投诉
- 说明：它不是已经坐实跑路，但最近窗口期的口碑已经不够干净，至少不该继续无脑列入头部推荐

## 最终建议

- 如果预算足、以稳定主用为先：优先看 `Nexitally / TAG / ImmTelecom`
- 如果想压低成本但不想碰纯肉鸡低价场：优先看 `iKuuu / V2NY / 魔戒`
- 如果你重视 AI、家宽出口、账号纯净度：`TAG` 优先级可以高于传统“看片型头部场”
- 如果你要打游戏、开黑、Discord 语音：对奶昔系、Amy 这类以 TCP/流媒体见长的场要降预期
- 如果你只打算买一个机场：不建议年付起手，先月付或按量
